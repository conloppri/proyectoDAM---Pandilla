# 📱 Pandilla - Proyecto Final DAM

¡Bienvenido/a al repositorio de **Pandilla**! Este proyecto es una aplicación móvil multiplataforma diseñada para la organización y gestión de grupos en tiempo real. Ha sido desarrollada como el **Proyecto Final del Grado Superior en Desarrollo de Aplicaciones Multiplataforma (DAM)**.

---

## 🚀 Características Principales

La aplicación permite a los usuarios crear grupos y colaborar de forma dinámica mediante las siguientes funcionalidades sincronizadas:

*   **🔐 Autenticación Segura:** Registro e inicio de sesión de usuarios controlado.
*   **📅 Calendario Compartido:** Gestión de eventos y fechas clave visibles para todo el grupo en tiempo real.
*   **📝 Notas Colaborativas:** Espacio común para redactar, editar y compartir anotaciones al instante.
*   **✅ Listas Dinámicas:** Creación de listas de tareas o elementos interactivos donde los cambios se reflejan inmediatamente para todos los miembros.

---

## 🛠️ Stack Tecnológico

*   **Frontend / Core:** [Dart](https://dart.dev/) & [Flutter](https://flutter.dev/) (Desarrollo multiplataforma).
*   **Backend / Cloud (BaaS):** [Firebase](https://firebase.google.com/)
    *   **Firestore:** Base de datos NoSQL para la sincronización en tiempo real.
    *   **Firebase Authentication:** Gestión y seguridad de las cuentas de usuario.
*   **IDE:** Android Studio.
*   **Control de Versiones:** Git & GitHub.
